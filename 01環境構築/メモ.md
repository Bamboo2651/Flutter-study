## 手順

### 1. Flutter SDK のインストール

1. [公式サイト](https://docs.flutter.dev/install/manual) から SDK をダウンロード
2. zip を解凍して好きな場所に保存する

   ```
   例: C:\SDK\flutter
   ```

   > ⚠️ パスにスペース・日本語が含まれていないか確認すること  
   > ⚠️ `C:\Program Files\` 以下には置かない

3. SDKは1つで複数プロジェクトから共用できる（プロジェクトと同じ場所に置く必要はない）

---

### 2. 環境変数（PATH）の設定

1. 「システム環境変数の編集」を開く
2. 「環境変数」をクリック
3. システム環境変数の `Path` を選択して「編集」
4. 以下を追加する

   ```
   C:\SDK\flutter\bin
   ```

5. コマンドプロンプトで確認

   ```bash
   flutter
   flutter doctor
   ```

> **PATHを通す意味**  
> どのフォルダにいても `flutter` コマンドが使えるようになる設定のこと。  
> SDKの場所をWindowsに教えているイメージ。

---

### 3. Android Studio のインストール

1. [Android Studio](https://developer.android.com/studio?hl=ja) をダウンロードしてインストール（全部 Next でOK）
2. 起動後、Plugins から **Flutter** をインストール
3. Projects 画面の「More Actions」→「SDK Manager」を開く
4. 「SDK Tools」タブで **Android SDK Command-line Tools** にチェックを入れる

> **注意**  
> Android Studio はコーディングに使わなくてもOK。  
> Android SDK の管理・エミュレータ作成のために必要なツール。  
> 普段のコーディングは VSCode で行う。

---

### 4. Android ライセンスの承認

コマンドプロンプトで以下を実行し、すべて `y` で進む。

```bash
flutter doctor --android-licenses
```

完了後、再度確認：

```bash
flutter doctor
```

---

### 5. Visual Studio のインストール（Windows Desktop 開発用）

> Android・Web のみ開発する場合はスキップしてOK。  
> Windows アプリも作りたい場合に必要。

1. [Visual Studio](https://visualstudio.microsoft.com/downloads/) から **Community（無料）** をダウンロード
2. インストーラーで **「Desktop development with C++」** にチェックを入れる
3. インストール完了後、`flutter doctor` で確認

> **Visual Studio と Visual Studio Code は別物！**  
> - Visual Studio Code → コードエディタ（普段使うやつ）  
> - Visual Studio → Windows アプリビルド用のC++コンパイラを含むツール群

---

### 6. VSCode の設定

以下の拡張機能をインストールする。

| 拡張機能 | 用途 |
|---------|------|
| Flutter | Flutter 開発サポート |
| Dart | Dart 言語サポート |

---

### 7. プロジェクトの作成

コマンドプロンプトでプロジェクトを作りたいディレクトリに移動してから実行：

```bash
flutter create プロジェクト名
```

> プロジェクト名は英数字（スネークケース推奨）  
> 例: `flutter create my_app`

## flutter doctor の見方

```
[√] → OK
[X] → エラー・未設定
[!] → 警告（動くが推奨設定ではない）
```

### 正常な状態の例（Android・Web のみの場合）

```
[√] Flutter
[√] Windows Version
[√] Android toolchain
[√] Chrome
[X] Visual Studio  ← Windows アプリ不要ならそのままでOK
[√] Connected device
[√] Network resources
```
