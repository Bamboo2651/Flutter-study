# 📁 プロジェクト構成

## フォルダ構成

```
lib/
├── 01_new/               # 新規プロジェクトの雛形
├── 02_Widget/            # Widgetの基礎
├── 03_Column_Row/        # Column・Rowで要素を並べる
├── 04_image/             # 画像の表示方法
├── 05_Container/         # Containerで装飾する
├── 06_StatelessWidget/   # StatelessWidgetの作り方
└── 07_Button/            # ボタンと関数の基礎
```

---

## 各フォルダの内容

| フォルダ | 内容 |
|---------|------|
| 01_new | flutter create で生成される初期プロジェクト |
| 02_Widget | Text・Center・Scaffold など基本Widgetの使い方 |
| 03_Column_Row | Column（縦並び）・Row（横並び）・mainAxisAlignment など |
| 04_image | Image.asset / network / memory / file の4種類の画像表示 |
| 05_Container | サイズ・色・padding・BoxDecoration で装飾する方法 |
| 06_StatelessWidget | stless スニペットを使ったWidgetの分割・作り方 |
| 07_Button | ElevatedButton・関数（普通・無名・アロー）の使い方 |

> 学習が進み次第、フォルダを追加していきます。
